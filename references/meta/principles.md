# UI 原则 / 定律 / 技巧 / 案例

> 规范层。跨维度的"**做什么**"知识——认知定律、UI 技巧、改版案例。与维度规范(color/font/icon/spacing/radius/border)的"**用什么值**"分离(Decision 2)。定律是多维度交叉的(如费茨同时影响尺寸/间距/圆角),故独立于此,不塞入维度文档。

## 一、认知与交互定律

| 定律            | 核心                        | 设计应用                           |
| --------------- | --------------------------- | ---------------------------------- |
| 费茨定律        | 目标越大、距离越近,命中越快 | 关键操作放大放近;热区 ≥ 44pt       |
| 米勒定律        | 短时记忆 7±2                | 导航/标签项 ≤ 7;信息分块           |
| 席克定律        | 选项越多,决策越久           | 减少选择,隐藏次级                  |
| 接近性(邻近性)  | 相关元素靠近即视为一组      | 用间距分组,非分割线                |
| 泰斯勒定律      | 复杂度守恒                  | 每处简化都把复杂度推到别处,需权衡  |
| 奥卡姆剃刀      | 如无必要勿增实体            | 删除优于添加                       |
| 防错原则        | 预防优于纠错                | 危险操作二次确认、不可逆操作加阻尼 |
| 冯·雷斯托夫效应 | 与众不同者被记住            | 关键项视觉隔离(唯一强调色)         |
| 蔡格尼克效应    | 未完成更被记得              | 进度条/未完成任务提升粘性          |
| 序列位置效应    | 首尾更被记住                | 首放导航,尾放行动                  |
| 共同区域原则    | 同一封闭区域视为一组        | 卡片/容器边界定义组                |
| 雅各布定律      | 用户习惯其他产品            | 走平台惯例,别发明交互              |
| **第 13 定律 · 动画动机** | 所有动画必须有动机(状态变化 / 空间引导 / 反馈),装饰性动画会降低可用性 | 无动机的动画直接砍掉;每段动画先回答"为什么动" |
| **第 14 定律 · 完整交互状态** | 交互元素必须覆盖 Loading / Empty / Error / Tactile Feedback 四类状态 | 缺任一类即"半成品",preview 阶段硬性失败 |

> 第 13 / 14 定律来源:taste-skill。第 13 定律与 [`dials.md`](./dials.md) MOTION_INTENSITY 联动:Level 1-3 严格只允许有动机动画;第 14 定律的 Tactile Feedback 与 [`vocabulary/micro-interactions.md`](../vocabulary/micro-interactions.md) 联动。

## 二、UI 技巧(48 项,按主题)

### 配色

- 6-3-1 配色比例(主 60 / 辅 30 / 强 10)
- 渐变背景减单调(会员卡/品牌强化)
- 高饱和色区分数据类目(后台图表)
- 颜色深浅构建层次(配合字重,不仅靠字号)
- 统一色调(一基础色调饱和度明度)
- 同色系深浅变化(温和配色)
- 颜色过渡变化(大面积平铺改渐变)
- 深色模式避免过多高饱和(仅重点文字用)

### 布局 / 留白

- 提高屏幕利用率(少滚动少导航)
- 双倍间距 + 双倍圆角(图标距边双倍)
- 视觉层级清晰(留白减负担)
- 卡片 F 型阅读(重点放扫描起点)
- 卡片间距留白(平衡秩序)
- 图片元素重叠 + 高斯模糊背景(空间感)
- 表单慎用分割线(改用背景/分组)
- 减少 banner 承载信息(简洁化重点)
- 优化交互路径(对角线布局、按钮放右下)
- 文本与昵称对齐(F 型动线)

### 图标 / 组件

- 图标 + 文字突出卖点
- 超椭圆(squircle)更舒适
- 操作类卡片优先按钮(非线性图标)
- 图标保持一致(风格/线宽/圆角)
- tab 栏差异化与趣味性

### 文字排版

- 视觉层级突出重要信息
- 文字切割与重组(动态感)
- 长文本适当变浅(减压迫)
- 简化样式与颜色(减杂乱)
- 数字特殊字体(价格/统计对比)
- 突出数据减少日期占比(健康/金融类)
- 加强对比突出重点
- 减少大色块干扰(清晰阅读)

### 质感 / 特效

- 干净阴影(微妙垂直偏移,顶光)
- 微妙纹理(阅读类纸张感)
- 毛玻璃卡片(柔和美感,减背景干扰)

### 情感化设计

- 强化头部氛围(客服/引导)
- 阅读类适当拟物(唤起纸书情感)
- 空页面情感化(幽默/温馨/激励)
- 弹窗强化情绪(庆祝/彩带/鼓励)
- 拟物化引导新人

### 数据可视化

- 强化提醒事项(限时/候补防误解)
- 增加优惠氛围强度(促转化)
- 金融数据图形化(折线感知波动)
- 测评信息可视化(雷达图整合)

### 导航交互

- 地图类卡片加视觉引导(地标元素)
- 可点击信息标签化(降认知)

## 三、改版案例(核心手法)

1. 视觉引导确立转化路径(IP/人物主体强化场景)
2. 榜单差异化(头部作者突出、排名逻辑清晰)
3. AI 问答 IP + 结构化(降冰冷感)
4. 潮流卡牌(明暗对比 + 纹理分区)
5. 金刚区业务前置(引导业务路径)
6. 产品图纯背景(避免花哨干扰)
7. 插画 / IP 强化品牌记忆(C 端)
8. 拟物化卡片(电子机票唤起期待)
9. 视觉降噪 + 信息重组(降认知负荷)
10. 拟态玻璃(智能家居场景融合)
11. 卡片高效聚焦(标签化紧凑,非松散列表)
12. 标签克制色彩(单主色突出选中)
13. 深色页避免大面积高饱和(小范围点缀)
14. 异形切割(热门话题打破矩形)
15. 深色模式 + 动态图(艺术类调性)
16. 列表亲密性 + 对比 + 呼吸感
17. 文本与昵称对齐(F 型动线)
18. 品牌调性强化(理念个性融入)
19. 动态渐变模糊背景联动(舞台质感)
20. 重色元素植入选空间层次(前后主次)
21. 卡片分层展示(移动端小屏清晰)
22. 优先级按钮(层级引导转化)
23. 个人中心背景不刺眼(低饱和渐变)
24. 商品纯色背景 + 抠图(0.5 秒捕捉)
25. 榜单前三差异化(强→弱动线)
26. 次功能入口用线性图标(克制统一)
27. 慎用黑色蒙版渐变(改近色调)
28. 降饱和明度凸显文字(卡片作中间层)
29. 电商产品图避免色彩杂乱
30. 多按钮并列区分重要性(搜索独立)
31. 合理视线流动引导(顺畅不跳)

## 四、平台补充规范(小红书 / 图文)

- 图片尺寸 1080×1440(3:4 竖图,小红书主流)
- 封面优先(决定点击率),文字大、主题明
- 轮播设计:首图抓眼、尾图引导互动
- 排版趋势:大字标题 + 留白 + 强对比
