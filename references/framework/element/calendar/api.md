# Calendar API

> `<el-calendar>` 的属性 / 事件 / 方法 / 插槽完整定义。用法见 [component.md](./component.md)。

## 属性(Props)

| 属性名                | 类型                | 默认值 | 说明                          |
| --------------------- | ------------------- | ------ | ----------------------------- |
| `model-value` / `v-model` | `Date`          | —      | 当前日期                      |
| `range`               | `Array<Date>`       | —      | 日期范围 [start, end]         |

## 事件(Events)

| 事件名    | 回调签名             | 说明              |
| --------- | -------------------- | ----------------- |
| `change`  | `(date: Date) => void` | 选中日期变化时 |
| `input`   | `(date: Date) => void` | (v-model 同步) |

## 方法(Methods)

| 方法名            | 签名                | 说明                       |
| ----------------- | ------------------- | -------------------------- |
| `selectDate`      | `(day: number) => void` | 选中某天(配合 ref)    |

## 插槽(Slots)

| 插槽名       | 作用域参数                | 说明                          |
| ------------ | ------------------------- | ----------------------------- |
| `date-cell`  | `{ date: Date, data: { type, isSelected, day } }` | 自定义日期单元格 |
| `header`     | —                         | 自定义头部                    |
