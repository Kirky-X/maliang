# Grid API(响应式栅格)

> 栅格 API 完整定义见 [../layout/api.md](../layout/api.md)。本文件仅补充响应式断点对象的字段说明。

## el-col 响应式属性对象结构

每个断点(`xs`/`sm`/`md`/`lg`/`xl`)可传入数字或如下对象:

| 字段名   | 类型     | 说明                |
| -------- | -------- | ------------------- |
| `span`   | `number` | 栅格占据列数        |
| `offset` | `number` | 栅格左侧间隔格数    |
| `pull`   | `number` | 栅格向左移动格数    |
| `push`   | `number` | 栅格向右移动格数    |

## 断点对照

| 断点 | 触发最小宽度 | 说明                       |
| ---- | ------------ | -------------------------- |
| `xs` | —            | 默认(手机竖屏)          |
| `sm` | `768px`      | 平板竖屏                   |
| `md` | `992px`      | 平板横屏 / 小笔记本        |
| `lg` | `1200px`     | 桌面                       |
| `xl` | `1920px`     | 大桌面                     |

> 响应式规则:某断点设置后,向更大断点继承,直到被覆盖。

## 常用响应式 span 对照(24 栅格)

| 目标列数 | span | xs | sm | md | lg  | xl  |
| -------- | ---- | -- | -- | -- | --- | --- |
| 1 列     | 24   | 24 | 24 | 24 | 24  | 24  |
| 2 列     | 12   | 24 | 12 | 12 | 12  | 12  |
| 3 列     | 8    | 24 | 12 | 8  | 8   | 8   |
| 4 列     | 6    | 24 | 12 | 8  | 6   | 6   |

## gutter 与 spacing token 建议

`<el-row :gutter>` 单位为 px,建议在项目中映射:

| gutter 值 | 近似 token      | 用途         |
| --------- | --------------- | ------------ |
| `8`       | `{spacing-xs}`  | 紧凑布局     |
| `12`      | `{spacing-sm}`  | 常规列表     |
| `20`      | `{spacing-lg}`  | 卡片栅格     |
| `24`      | `{spacing-xl}`  | 宽松分栏     |

> 完整属性表见 [../layout/api.md](../layout/api.md)。
